# O17e – Diffraction

The experiment investigated diffraction and interference of monochromatic laser light through single and double slits in both the Fraunhofer and Fresnel regimes using CCD-based intensity measurements and model fitting.

## Focus
- single-slit and double-slit diffraction
- Fraunhofer and Fresnel regimes
- extraction of slit width and slit separation
- spatial and Fourier-domain analysis of intensity profiles

## Files
- `report.pdf` – final lab report
- `Notebook.zip` – measurement and analysis files
