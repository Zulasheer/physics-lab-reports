# Physics Lab Reports

This repository contains a collection of laboratory reports, experimental data, and supporting files from my university physics coursework.

## Contents

Each experiment is organized in its own folder and includes:
- the final report in PDF format
- measurement or analysis data files
- a short README describing the experiment

## Topics

- Geometrical optics
- Interference and diffraction
- Polarisation
- Atomic spectra
- AC circuits and resonance
- Semiconductor devices
- Coupled oscillations

## Purpose

This repository is intended as a structured portfolio of my experimental physics work, including data analysis, interpretation of results, and scientific report writing.
