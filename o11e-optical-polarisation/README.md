# O11e – Optical Polarisation

The experiment investigated how the transmitted intensity of polarized light changes with analyser angle for different quarter-wave plate orientations and also analyzed the polarization state of unknown optical components.

## Focus
- Jones calculus
- quarter-wave plate behavior
- analyser-angle dependent intensity
- identification of unknown polarization states
- qualitative and Stokes-parameter based analysis

- `report.pdf` – final lab report
